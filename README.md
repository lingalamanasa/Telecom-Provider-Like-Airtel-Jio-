## 📖 About the Project

**Stackly Telecom** is a fully responsive, modern telecom provider website inspired by industry leaders like **Airtel** and **Jio**. Built with pure HTML, CSS, and JavaScript, it showcases a premium 5G telecom brand with multi-page navigation, animations, and a polished dark-mode design.

---

## ✨ Features

- 🌑 **Dark Mode UI** with glassmorphism effects
- 📱 **Fully Responsive** — works on all screen sizes
- ⚡ **Animated Hero Section** with dynamic stats
- 📶 **Services Page** — Mobile, Fiber, Enterprise & Roaming plans
- 👥 **About Us Page** — Team, Milestones, Awards & Values
- 📝 **Blog Page** — Telecom news and articles
- 📬 **Contact Page** — with form and interactive map
- 🔐 **Login / Register Pages** — styled auth forms
- 📊 **Admin Dashboard** — subscriber, revenue & network stats
- 🔔 **404 Error Page** — custom branded error page
- 🎞️ **Scroll Animations** using Intersection Observer API

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/lingalamanasa/Telecom-Provider-Like-Airtel-Jio-.git
   ```

2. **Navigate into the project**
   ```bash
   cd Telecom-Provider-Like-Airtel-Jio-
   ```

3. **Open in browser** — open `index.html` directly, or use the VS Code **Live Server** extension.
